# 更新日志 / Changelog

## **[1.0.4] - 2026-06-17**
新增 / Added
-  器件拓扑图 - 可视化展示原理图中器件之间的连接关系网络
-  拓扑图默认过滤 GND 网络，避免干扰网络可视化
-  拓扑图支持点击引脚高亮同名网络
-  拓扑图支持按网络筛选过滤显示内容

优化 / Optimized
-  连接关系图算法优化 - 改进电源和 GND 网络的识别逻辑，减少误分类
-  多器件场景下的渲染性能优化，提升流畅度
-  更新扩展打包构建流程，确保路径格式兼容性
